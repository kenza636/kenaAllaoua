📄 Description des Fichiers

📌 index.html

Rôle: Page principale de l'application
Contient: Formulaire de connexion, logo, bouton retour
À ouvrir: Directement dans le navigateur
Liens vers: signup.html, forgot-password.html, dashboard.html

📌 css/styles.css

Rôle: Tous les styles de l'application
Contient: Variables CSS, designs, animations, responsive
Utilisé par: Toutes les pages HTML
Points clés: Couleurs, dégradés, transitions

📌 js/script.js

Rôle: Logique JavaScript du site
Gère: Validation, toggling, événements, messages
Utilisé par: Toutes les pages HTML
Fonctions clés: togglePassword(), showMessage(), validation

📌 pages/signup.html

Rôle: Formulaire d'inscription
Champs: Nom, Email, Mot de passe, Confirmation
Accès: Lien "Créer un compte pro" depuis index.html
Utilise: css/styles.css pour le design

📌 pages/forgot-password.html

Rôle: Récupération du mot de passe
Demande: Email de l'utilisateur
Accès: Lien "Mot de passe oublié ?" depuis index.html
Action: Envoie un lien de réinitialisation (simulation)

📌 pages/dashboard.html

Rôle: Tableau de bord après connexion
Affiche: Sidebar, menu, statistiques, cartes d'info
Accès: Programmé après une connexion réussie
Contient: Navigation principale, profil utilisateur

🔄 Flux de Navigation

INDEX.HTML (Connexion)
    ├─→ Cliquer "Créer compte pro" → signup.html
    ├─→ Cliquer "Mot de passe ?" → forgot-password.html
    └─→ Cliquer "Se connecter" → dashboard.html

SIGNUP.HTML (Inscription)
    └─→ Compte créé → retour à index.html

FORGOT-PASSWORD.HTML (Récupération)
    └─→ Email envoyé → retour à index.html

DASHBOARD.HTML (Tableau de bord)
    └─→ Cliquer "Déconnexion" → index.html
